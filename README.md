# MongoCrud
CRUD api Based on randomuser.me/api model

> [!NOTE]
> If you want to utilize the POST endpoint, you can copy the body from https://randomuser.me/api/?results=1 ;it's 1:1 .  
> You can also POST multiple results ( ?results=x )
